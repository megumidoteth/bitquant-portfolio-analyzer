# bitquant-portfolio-analyzer
this is a community-built tool that lets you analyze your crypto holdings using AI quantitative analysis. It is accessible to everyone and built on opengradient

how it works:

user enters the assets and amounts and the AI runs a full quant breakdown and returns like

portfolio health score: how strong users' overall portfolio is

risk Level: low, medium or high

diversification score: how spread out your exposure is

per-asset insights: bullish, bearish, neutral or warning for each coin

top recommendation: one clear and actionable step you should take

verification hash: every analysis is stamped onchain via OpenGradient
